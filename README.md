# Stylish VPN App
## 概要
SwiftUIでスタイリッシュなUIアニメションのVPNアプリを作成してみました。
SwiftUIのCapsuleを用いてStart/Stopボタンを描画したりタップでon/offアニメーションをつけたりするUIサンプルコードです。
